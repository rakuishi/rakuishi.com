頻繁に使うのですが、これまで何回もコードを忘れて検索しているので、コピペ用としてブログに書いておきます。

**インジケータを表示する**
```
[UIApplication sharedApplication].networkActivityIndicatorVisible = YES;
```

**インジケータを非表示にする**
```
[UIApplication sharedApplication].networkActivityIndicatorVisible = NO;
```
